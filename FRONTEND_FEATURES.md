# 🎫 票务系统 - 前端功能文档

## 📱 新增前端页面

### 1. **注册页面** (`/register`)
- 新用户可以创建账户
- 验证用户名、邮箱唯一性
- 密码长度验证（至少6位）
- 密码确认验证
- 注册成功后自动登录

**访问方式**: 登录页面 → "立即注册" 链接

### 2. **个人资料页面** (`/profile`)
- **基本信息标签**:
  - 修改姓名
  - 修改邮箱
  - 用户名不可修改
  
- **修改密码标签**:
  - 输入当前密码验证
  - 设置新密码
  - 密码确认验证

**访问方式**: 导航栏 → 用户名 → 个人资料

### 3. **收藏页面** (`/favorites`)
- 显示所有收藏的活动
- 活动卡片包含：
  - 活动图片
  - 活动名称、地点、时间
  - "查看详情" 按钮
  - "取消收藏" 按钮
- 空状态提示

**访问方式**: 导航栏 → ❤️ 收藏

### 4. **优惠券页面** (`/coupons`)
- 显示所有可用优惠券
- 优惠券信息：
  - 优惠券代码
  - 折扣类型和金额
  - 最低消费要求
  - 剩余使用次数
  - 有效期
  
- **验证面板**:
  - 输入优惠券代码
  - 输入订单金额
  - 实时验证结果显示

**访问方式**: 导航栏 → 🎟️ 优惠券

## 🎭 活动详情页面增强 (`/events/:id`)

### 新增功能：
1. **收藏按钮**
   - 显示收藏状态（❤️ 已收藏 / 🤍 收藏）
   - 点击切换收藏状态

2. **评分显示**
   - 显示平均评分（★）
   - 显示评价数量

3. **评论区域**
   - **评论表单**:
     - 5星评分选择
     - 评价内容输入框
     - 提交按钮
   
   - **评论列表**:
     - 用户名
     - 星级显示
     - 评价内容
     - 发布时间
     - 仅购票用户可评价

## 💳 结算页面增强 (`/checkout`)

### 新增功能：
1. **优惠券输入**
   - 输入优惠券代码
   - 验证按钮
   - 验证结果提示

2. **价格计算**
   - 显示小计
   - 显示优惠金额（如有）
   - 显示最终总价

3. **支付按钮**
   - 显示最终支付金额
   - 使用优惠券后自动更新

## 📋 订单页面增强 (`/orders`)

### 新增功能：
1. **订单状态**
   - 已完成（绿色标签）
   - 已取消（红色标签）

2. **取消订单**
   - 已完成订单显示"取消订单"按钮
   - 24小时内可取消
   - 取消后显示退款金额

3. **优惠券显示**
   - 显示优惠金额（如有）

4. **订单卡片**
   - 已取消订单显示半透明效果
   - 显示退款信息

## 🧭 导航栏更新

新增导航链接：

- 🎟️ 优惠券 → `/coupons`
- 👤 用户名 → `/profile`

## 🔐 认证流程

### 登录流程
1. 访问应用 → 显示登录页面
2. 输入用户名和密码
3. 点击"登录"
4. 成功后进入活动列表

### 注册流程
1. 登录页面 → 点击"立即注册"
2. 填写注册表单
3. 点击"注册"
4. 成功后自动登录

## 🎯 使用流程示例

### 完整购票流程
1. 登录系统
2. 浏览活动列表
3. 点击活动进入详情页
4. **可选**: 收藏活动 (❤️)
5. **可选**: 查看评价和评分
6. **可选**: 添加评价
7. 选择座位
8. 进入结算页面
9. **可选**: 输入优惠券代码验证
10. 确认支付
11. 支付成功后查看订单

### 优惠券使用流程
1. 导航栏 → 🎟️ 优惠券
2. 查看可用优惠券
3. 输入优惠券代码和订单金额
4. 点击"验证"
5. 查看优惠结果
6. 在结算页面使用该优惠券
