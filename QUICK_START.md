# 🚀 快速开始指南

## 📦 系统要求

- Docker & Docker Compose
- 浏览器 (Chrome/Firefox/Safari/Edge)
- 网络连接

## 🎯 5分钟快速启动

### 1. 启动容器
```bash
cd projects/s3
docker-compose up -d
```

### 2. 等待服务启动
```bash
# 检查容器状态
docker ps | grep s3

# 应该看到:
# s3_server_1  Up  0.0.0.0:8001->8000/tcp
# s3_client_1  Up  0.0.0.0:5174->5173/tcp
```

### 3. 访问应用
打开浏览器访问: **http://localhost:5174**

### 4. 登录系统
使用测试账号登录:
- 用户名: `zhangsan`
- 密码: `123456`

## 🐛 故障排除

### 前端无法加载
```bash
# 检查容器
docker ps | grep s3_client

# 查看日志
docker logs s3_client_1
```

### 后端 API 错误
```bash
# 检查后端容器
docker ps | grep s3_server

# 查看日志
docker logs s3_server_1

# 重置数据
curl -X POST http://localhost:8001/api/admin/reset
```

### 端口被占用
```bash
# 修改 docker-compose.yml 中的端口
# 然后重启容器
docker-compose down
docker-compose up -d
```
